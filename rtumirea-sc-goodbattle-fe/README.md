# Good Battle

Платформа для баттлов по программированию в реальном времени.

## Идея

Участники решают простые задачи на скорость, при этом **код каждого виден всем в реальном времени** — это создаёт напряжение и соревновательный дух.

**Организатор** создаёт комнату, выбирает задачи и управляет процессом (старт, пауза, завершение), но сам не участвует — он видит код всех участников и следит за ходом баттла.

**Участники** присоединяются по коду комнаты, пишут решения и наблюдают за прогрессом соперников.

## Авторизация

Регистрация и вход по email/паролю. Авторизация необходима для создания комнат, участия в баттлах и доступа к истории.

## Страницы

- **Главная** `/` — выбор роли (организатор / участник), вход в комнату по коду
- **Регистрация** `/register` — создание аккаунта
- **Авторизация** `/login` — вход в существующий аккаунт
- **Создание комнаты** `/create-room` — настройки баттла: разрешённые языки, лимит времени на задачу, выбор задач из списка
- **Комната** `/rooms/$roomId` — основной экран баттла:
  - Текущая задача (общая для всех)
  - Таймер обратного отсчёта
  - Редакторы кода всех участников в реальном времени
  - Для организатора: управление процессом (старт, пауза, следующая задача, завершение)
  - Для участника: свой редактор + просмотр кода соперников
- **Результаты** — диалог по окончании баттла с топом участников
- **Мои баттлы** `/battles` — история баттлов с табами «Участник» / «Организатор»
- **Профиль** `/profile` — информация о пользователе, статистика

## Стек

| Категория         | Технология                                          |
| ----------------- | --------------------------------------------------- |
| Фреймворк         | React 19, TypeScript                                |
| Сборка            | Vite 7                                              |
| Роутинг           | TanStack Router (file-based)                        |
| Стили             | Tailwind CSS 4                                      |
| UI-компоненты     | shadcn/ui (стиль radix-nova, базовый цвет zinc)     |
| Редактор кода     | CodeMirror 6 (@uiw/react-codemirror, тема One Dark) |
| Ресайз панелей    | react-resizable-panels                              |
| Иконки            | Lucide React                                        |
| Пакетный менеджер | pnpm                                                |

## Структура проекта

```
src/
├── api/                 # API-клиент, React Query hooks и query options
│   ├── __generated__/   # Автогенерация
│   ├── hooks/
│   ├── client.ts
│   ├── index.ts
│   └── query-client.ts
├── components/
│   ├── header.tsx       # Общий хедер
│   ├── battle/          # Компоненты страницы баттла
│   │   ├── battle-context.tsx       # React Context со стейтом баттла
│   │   ├── battle-results-dialog.tsx # Диалог результатов
│   │   ├── code-editor.tsx          # Обёртка CodeMirror
│   │   ├── countdown-timer.tsx      # Таймер обратного отсчёта
│   │   ├── editor-grid.tsx          # Сетка редакторов (resizable, табы участников)
│   │   ├── organizer-controls.tsx   # Кнопки управления баттлом
│   │   ├── participant-editor.tsx   # Карточка участника (редактор + язык + запуск)
│   │   └── task-panel.tsx           # Панель задачи (таймер, описание, примеры)
│   └── ui/              # shadcn-компоненты
│       └── index.ts     # Barrel-экспорт UI-компонентов
├── icons/               # SVG-иконки и логотип (vite-plugin-svg-sprite)
├── lib/
│   └── utils.ts         # Утилиты (cn — объединение classNames)
├── routes/              # Файловый роутинг TanStack Router
│   ├── __root.tsx       # Корневой layout + error boundary + desktop/mobile split
│   ├── (auth)/          # Публичная зона (/login, /register)
│   └── (app)/           # Приватная зона (/, /create-room, /battles, /profile, /rooms/$roomId)
├── stores/              # Zustand-сторы
├── app.tsx              # Инициализация роутера
├── main.tsx             # Точка входа
├── routeTree.gen.ts     # Автогенерация
└── index.css            # Глобальные стили, CSS-переменные, тема
```
