# Good Battle

Платформа для баттлов по программированию в реальном времени.

## Идея

Участники решают простые задачи на скорость, при этом **код каждого виден всем в реальном времени** — это создаёт напряжение и соревновательный дух.

**Организатор** создаёт комнату, выбирает задачи и управляет процессом (старт, пауза, завершение), но сам не участвует — он видит код всех участников и следит за ходом баттла.

**Участники** присоединяются по коду комнаты, пишут решения и наблюдают за прогрессом соперников.
